VoiceJournal

Solo Android journaling app with voice-to-text transcription and optional cloud backups.

What it does

  • Records voice entries and transcribes them into text.
  • Supports keyword tagging and calendar-based browsing.
  • Integrates Google Drive API for optional backups.

Tech

Kotlin Jetpack Compose Speech-to-text Google Drive API
